# MASS Entity Performance Test (Unreal Engine 5.7.3)

## 📌 Description

This project was created by following this YouTube tutorial:  
https://www.youtube.com/watch?v=f9q8A-9DvPo

The project was developed using **Unreal Engine 5.7.3** and the **MASS framework**.

## 🚀 Features

- Generates **10,000 entities**
- All entities move in real time
- Maintains **100+ FPS**
- Demonstrates high-performance large-scale simulation
